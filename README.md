# 🍽 Zomato Restaurant Analysis using NLP and Unsupervised Machine Learning

## 📌 Project Overview
Zomato is a popular restaurant discovery and food delivery platform that provides information about restaurants, cuisines, pricing, ratings, and customer reviews. With thousands of restaurants and large volumes of customer feedback, analyzing this data can provide valuable insights into restaurant performance and customer preferences.

This project analyzes Zomato restaurant data using **Natural Language Processing (NLP)** and **Unsupervised Machine Learning techniques** to discover patterns in restaurant reviews and metadata.

The goal of this project is to cluster restaurants based on their characteristics and customer feedback to identify meaningful groups and trends in the restaurant industry.

---

## 🎯 Objectives
- Perform **Exploratory Data Analysis (EDA)** on restaurant metadata and reviews.
- Apply **Natural Language Processing (NLP)** to clean and process textual reviews.
- Transform review text into numerical features using **TF-IDF vectorization**.
- Reduce dimensionality using **Principal Component Analysis (PCA)**.
- Implement clustering algorithms to group restaurants with similar characteristics.
- Generate insights that help understand **customer behavior and restaurant performance**.

---

## 📂 Dataset
This project uses two datasets:

### 1️⃣ Restaurant Metadata Dataset
Contains restaurant information such as:
- Restaurant Name
- Cost
- Cuisine Type
- Collections
- Timings
- Zomato Links

### 2️⃣ Restaurant Reviews Dataset
Contains customer review information including:
- Reviewer Name
- Review Text
- Rating
- Review Metadata
- Time of Review
- Number of Pictures
